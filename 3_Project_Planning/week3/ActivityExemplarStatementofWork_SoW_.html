Description of change

Purpose: The goal of this Statement of Work (SoW) is to successfully configure and install new inventory and fulfillment tracking software for the Office Green team.

Scope / major project activities:

  • Install and configure the software on all equipment that employees will use to manage inventory and schedule deliveries (includes auditing existing Office Green software and developing optimizations)
  • Create a training manual and maintenance guide for the software
  • Train two office green employees

Out-of-scope activities:

  • Ongoing maintenance of the software (beyond the contract dates)
  • Ongoing training of employees (beyond the contract)

Deliverables:

  • Installation and configuration of software on all Office Green devices
  • Training manual and maintenance guide

Major Milestones:

  • Milestone 1: Software configuration
  • Milestone 2: Software installation
  • Milestone 3: Training manual and maintenance guide
